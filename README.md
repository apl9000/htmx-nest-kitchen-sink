A kitchen sink for building UI apps with NestJS, HTMX, Handlebars, TailwindCSS and few other utensils.

[demo](https://stingray-app-dkeuh.ondigitalocean.app/)

### Getting Started
Fork the project, install and start developing. 

#### Installation

```bash
$ npm install
```

#### Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run start:prod
```

#### Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
